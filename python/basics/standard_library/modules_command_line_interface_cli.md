# Python Modules with Command-Line Interface (CLI)

The following Python standard library modules provide a command-line interface (CLI). You can run them directly from the terminal using `python -m <module>` or similar commands.

---

| Module               | Description / Example CLI Usage                      |
|----------------------|-----------------------------------------------------|
| ast                  | Parse and analyze Python code: `python -m ast`      |
| asyncio              | Run asyncio event loop: `python -m asyncio`         |
| base64               | Encode/decode data: `python -m base64`              |
| calendar             | Print calendars: `python -m calendar`               |
| code                 | Interactive interpreter: `python -m code`           |
| compileall           | Byte-compile files: `python -m compileall <dir>`    |
| cProfile             | Profile programs: `python -m cProfile <script>`     |
| dis                  | Disassemble bytecode: `python -m dis <script>`      |
| doctest              | Run doctests: `python -m doctest <script>`          |
| encodings.rot_13     | Encode/decode rot13: `python -m encodings.rot_13`   |
| ensurepip            | Install pip: `python -m ensurepip`                  |
| filecmp              | Compare files: `python -m filecmp`                  |
| fileinput            | Process input files: `python -m fileinput`          |
| ftplib               | FTP client: `python -m ftplib`                      |
| gzip                 | Compress/decompress files: `python -m gzip`         |
| http.server          | Start HTTP server: `python -m http.server`          |
| idlelib              | Launch IDLE: `python -m idlelib`                    |
| inspect              | Inspect objects: `python -m inspect`                |
| json                 | JSON pretty print: `python -m json.tool`            |
| mimetypes            | Guess MIME types: `python -m mimetypes`             |
| pdb                  | Debugger: `python -m pdb <script>`                  |
| pickle               | Serialize/deserialize: `python -m pickle`           |
| pickletools          | Analyze pickle bytecode: `python -m pickletools`    |
| platform             | System info: `python -m platform`                   |
| poplib               | POP3 client: `python -m poplib`                     |
| profile              | Profile programs: `python -m profile <script>`      |
| pstats               | Profile stats: `python -m pstats <profile_file>`    |
| py_compile           | Compile files: `python -m py_compile <file>`        |
| pyclbr               | Module browser: `python -m pyclbr <module>`         |
| pydoc                | Documentation: `python -m pydoc <module>`           |
| quopri               | Encode/decode quoted-printable: `python -m quopri`  |
| random               | Generate random numbers: `python -m random`         |
| runpy                | Run modules: `python -m runpy <module>`             |
| site                 | Site info: `python -m site`                         |
| sqlite3              | SQLite shell: `python -m sqlite3`                   |
| symtable             | Symbol table info: `python -m symtable`             |
| sysconfig            | System config info: `python -m sysconfig`           |
| tabnanny             | Check indentation: `python -m tabnanny <file>`      |
| tarfile              | Tar archive operations: `python -m tarfile`         |
| this                 | Print Zen of Python: `python -m this`               |
| timeit               | Time code execution: `python -m timeit`             |
| tokenize             | Tokenize source: `python -m tokenize <file>`        |
| trace                | Trace execution: `python -m trace <script>`         |
| turtledemo           | Run turtle demos: `python -m turtledemo`            |
| unittest             | Run unit tests: `python -m unittest`                |
| uuid                 | Generate UUIDs: `python -m uuid`                    |
| venv                 | Create virtual env: `python -m venv <env_dir>`      |
| webbrowser           | Open URLs: `python -m webbrowser <url>`             |
| zipapp               | Manage zip archives: `python -m zipapp <dir>`       |
| zipfile              | Zip file operations: `python -m zipfile`            |
